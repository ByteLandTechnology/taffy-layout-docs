间距 (Gap)

定义行与列之间的间隔。

gap 属性定义了 Flexbox 和 Grid 布局中行与行、列与列之间的间距（沟槽）。它是 CSS 中 row-gap 和 column-gap 的简写，在 Taffy 中表示为 Size<LengthPercentage>。

取值

gap 接受一个包含 width（列间距）和 height（行间距）的 Size 对象，通常以像素或百分比为单位。

属性描述
width行内子元素之间的空间（列间距）。
height行与行之间的空间（行间距）。
