スティッキーフッター

コンテンツが短い場合、フッターは下部に固定されます。コンテンツが増えると、フッターは自然に下に移動します。

┌────────────────────────────┐
│ Header                     │
├────────────────────────────┤
│ Content (flex: 1)          │
├────────────────────────────┤
│ Footer                     │
└────────────────────────────┘

主要なルール

  • 親: flexDirection: column
  • コンテンツ: flexGrow: 1
