スタイリング

Taffy のスタイリングプロパティに関する包括的なガイド。

Taffy のスタイリング API は CSS をモデルにしています。以下は、サポートされているすべてのプロパティの分類リストです。

レイアウトモード

ノードの動作を定義するコアプロパティ。

プロパティ説明
displayFlex（デフォルト）、Grid、または None。
positionRelative（フロー）または Absolute（オーバーレイ）。
overflowVisible、Hidden、または Scroll。
insetポジショニング用の top、bottom、left、right。

サイズと間隔

寸法と間隔を制御します。

プロパティ説明
size / minSize / maxSize幅と高さの制御。
aspectRatio幅と高さの比率。
margin外側の余白。
padding内側の余白。
border枠線の幅（スペースのみ）。
gapFlex/Grid アイテム間の間隔。

Flexbox レイアウト

1D レイアウトのプロパティ。

プロパティ説明
flexDirectionRow または Column の方向。
flexWrapWrap または NoWrap。
flexBasis初期メインサイズ。
flexGrow伸長係数。
flexShrink収縮係数。
justifyContentメイン軸の整列。
alignItemsデフォルトのクロス軸の整列。
alignSelfアイテムの整列オーバーライド。
alignContent折り返し行の整列。

Grid レイアウト

2D レイアウトのプロパティ。

プロパティ説明
gridTemplate列と行を定義します。
gridColumn列方向へのアイテムの配置。
gridRow行方向へのアイテムの配置。
gridAutoFlow自動配置アルゴリズム。
