はじめに

Taffy Layout へようこそ！このガイドでは、高性能なレイアウトエンジンの使い方を説明します。

クイックリンク

  • インストール - 環境に Taffy をセットアップします。
  • クイックスタート - 数分で独自のレイアウトを作成します。
  • ノードツリーの構築 - レイアウトツリーの構築方法を学びます。
  • レイアウトの計算 - レイアウト計算プロセスをマスターします。
  • 設定 - ニーズに合わせて Taffy を微調整します。

なぜ Taffy なのか？

Taffy は、クロスプラットフォーム開発者にとって最も高速で信頼性の高いレイアウトエンジンとなるよう設計されています。ゲーム UI、独自のモバイルフレームワーク、または高性能な Web コンポーネントを構築している場合でも、Taffy は必要な標準準拠の Flexbox および Grid ツールを提供します。
