なぜ Taffy か

驚異的なスピード

Rust で書かれ、WebAssembly にコンパイルされています。最大の効率と最小のレイテンシを実現するようベンチマークされています。

Flexbox & Grid

CSS Flexbox と CSS Grid レイアウトアルゴリズムの両方を完全かつ堅牢にサポートし、ピクセルパーフェクトな結果を提供します。

モダンな標準

モダンな CSS レイアウト仕様に完全に準拠しており、Web 開発者にとって馴染み深いものになっています。

マルチプラットフォーム

どこでも動作します。ゲームエンジン、デスクトップアプリ、モバイルフレームワーク、高性能 Web アプリに最適です。
