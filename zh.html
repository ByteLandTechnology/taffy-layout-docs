Taffy Layout

Taffy Layout高性能布局引擎

为 Web、移动端及更多场景打造的灵活、高性能 Flexbox 和 Grid 布局引擎。基于 Rust 和 WASM 构建。

  • 无需 DOM 的弹性盒子与网格布局。
  • 为自定义渲染器提供快速、确定性的布局。
  • 为真实世界 UI 引擎打造的便携 API。

为什么选 Taffy

极速性能

使用 Rust 编写并编译为 WebAssembly。经过基准测试，确保最大效率和极低延迟。

Flexbox 与 Grid

完整且稳健地支持 CSS Flexbox 和 CSS Grid 布局算法，确保像素级精确的结果。

现代标准

完全符合现代 CSS 布局规范，让任何 Web 开发者都能快速上手。

多平台支持

随处运行。非常适合游戏引擎、桌面应用、移动框架以及高性能 Web 应用。
